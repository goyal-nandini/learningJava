##  Why do we need this?

Because **objects** can:

* be used in `ArrayList`, HashMap, etc.
* have methods
* store null
* be passed where objects are required

Primitive `int` cannot do these.

##  One ultra-simple example

```java

Integer b = 5;         // object containing 5
```

Memory:

```
a = 5                (stored directly in stack)
b -> [value = 5]     (object stored in heap)
```
